# 目录结构

```
├── align           对齐结果
│   ├── 223_clean.txt
│   ├── 224_clean.txt
│   ├── ...
│   └── {音频编号}_clean.txt
├── hyp.txt         // 模型预测结果
├── ref.txt         // 标准音频序列
```

# 对齐结果 

以`223_clean.txt`举例:

```
4.94 5.27 h
5.27 5.35 ao_4
5.35 5.51 k
31.85 31.93 ai_3
31.93 32.01 d
32.01 32.09 a_4
32.09 32.33 er_4
32.33 32.46 w
32.46 32.54 u_2
54.20 54.32 g
```

整个文件以行为单位，按时间顺序出现来代表对齐结果。每行的格式为:
```
开始时间(s) 结束时间(s) 当前音素名称
```

拼音的音调用数字代表:

- 0 轻声
- 1-4 一到四声

# hyp.txt 和 ref.txt的格式

```
音频编号   (制表符)   音素1 音素2 ... 音素n (换行)
```
